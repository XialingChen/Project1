Compiler

A compiler is a computer program (or a set of programs) that transforms source code written in a programming language (the source language) into another computer language (the target language), with the latter often having a binary form known as object code.

Here are examples of compilers of some popular programming languages:

Programming Language Compiler
BASIC BaCon, BAIL, BlitzMax, Chipmunk Basic, CoolBasic, DarkBASIC, Gambas, QB64
C AMPC, Aztec C, Amsterdam Compiler Kit, CCS C Compiler, Ch, Clang, CodeWarrior, PGCC
C++ C++ Builder, Turbo C++ Explorer, C++ Compiler, CINT, Borland C++, GCC, KAI C++ Compiler
C# Visual C#, Visual C# Express, Mono, Portable.NET, SharpDevelop, Roslyn
Java Roslyn, gcj, javac, S.N Java Compiler, ECJ (Eclipse Compiler for Java), Jikes, Power J
Python Psyco, Cython, Shed Skin, PyPy, Nuitka
